# ClariFi AI

ClariFi AI is a Decision Intelligence System that converts unstructured financial data (SMS, bank logs, CSV files) into actionable insights and recommendations.

## Features

* Multi-source input (SMS, CSV, bank statements)
* Spending insights and visualizations
* Pattern detection (subscriptions, duplicates, habits)
* Actionable recommendations with estimated savings

## Working

User Input → Data Processing → AI Analysis → Decisions → Dashboard Output

## Tech Stack

* Frontend: React / Next.js
* Backend: FastAPI / Node.js
* AI: LLM APIs
* Data Processing: Python (Pandas)

## Setup

```bash
git clone https://github.com/your-username/clarifi-ai.git
cd clarifi-ai
npm install
npm run dev
```

## Conclusion

ClariFi AI transforms raw financial data into clear, actionable decisions that help users save money.
